## What to Measure Going Forward

### After every 5-10 sessions, check:

```bash
# Engram exploration ratio (should drop from 29%)
engram stats --project monra-app

# Engram per-session costs (should see lower input tokens)
engram costs --limit 10

# Engram cache efficiency (should improve)
engram insights

# Claude-mem observation growth
curl -s http://127.0.0.1:37777/api/stats

# Noodlbox usage growth
engram artifacts --type api_call --project monra-app --limit 50
```

### How each system contributes:

- **Engram brief** → reduces exploration (agent knows key files upfront)
- **Engram PreToolUse hook** → reduces re-discovery (agent gets file history at read time)
- **Claude-mem** → reduces re-explanation (agent recalls past decisions/bugfixes)
- **Noodlbox** → reduces code navigation (agent finds functions via semantic search)
